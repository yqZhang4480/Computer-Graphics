Computer Graphics
=======
此repo包含计算机图形学的一些练习代码。<br>

开发环境
-------
* 运行环境：Windows，Visual Studio 2019
* 需要 EasyX 图形库，本项目使用版本：20200520（beta）

说明
-------
* 解决方案内包含多个VS项目，将欲执行项目设为启动项目即可运行。
